# Restful API Django

REST란? REpresentational State Transfer

  • HTTP를 이용해 통신하는 네트워크상에서 정한 약속
  • 분산 하이퍼미디어 시스템을 위한 소프트웨어 설계 형식
  • 자원을 대표하는 단어 / 식별자로 자원의 상태를 전송하는 방법
  • 자원을 이름으로 구분하여 상태를 전송하는 방법
  • 하위 호환을 깨뜨리지 않고 독립적 발전할 수 있게 도와주는 방법

REST 필요충분조건?

  • Server - Client
  • STATELESS
  • Cache
  • Uniform Interface
  • Layered System
  • Code-On-Demand
